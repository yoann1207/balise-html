Carte Mentale des Balises HTML

Cette carte mentale présente les balises HTML de base organisées par catégories. Passez votre souris sur les cartes pour les voir s'animer et explorez les différentes balises essentielles au HTML.

Structure de base

<head>

<body>

<title>
Titre de la page (onglet du navigateur)
<meta>
Métadonnées (encodage, viewport, etc.)
<link>
Liaison avec fichiers externes (CSS)
<style>
Styles CSS intégrés
<script>
Code JavaScript
Textes
<h1> à <h6>

<p>
Paragraphe
<span>
Conteneur inline générique
<br>
Saut de ligne
<hr>
Ligne horizontale
Contenu
<div>
Conteneur générique
<main>
Contenu principal
<section>

<article>
Contenu autonome
<header>
En-tête de section
<footer>
Pied de page
Médias
<img>
Image
<audio>
Contenu audio
<video>
Contenu vidéo
<figure>

<tr>

<form>

<textarea>

<select>

<button>

<label>
